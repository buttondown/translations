This repository contains translation files for [Buttondown](https://buttondown.com)'s subscriber-facing UI.

The `translations.json` file contains all translatable strings used in Buttondown's subscriber-facing interface, organized by language code. Each entry maps an English string to its translations in supported languages.

Supported languages include:

| Language            | Code  |
| ------------------- | ----- |
| Czech               | cs-CZ |
| German              | de    |
| English             | en    |
| Spanish             | es    |
| French              | fr    |
| Dutch               | nl-NL |
| Polish              | pl-PL |
| Portuguese (Brazil) | pt-BR |
| Turkish             | tr    |
